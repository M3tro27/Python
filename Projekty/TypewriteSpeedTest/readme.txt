TEST RYCHLOSTI PSANÍ


Obecné informace:

	- Program zobrazí větu a uživatel ji opíše do prázdného řádku.

	- Poté, co uživatel ukončí opis klávesou Enter, program vyhodnotí rychlost,
	  přesnost a wpm (words per minute).

	- Program disponuje dosavadní databází 29 vět, ze které náhodně vybírá.


Plány do budoucna:

	- Rozšíření databáze.

	- Přidání dalších možností jak vybírat věty, zda věty z databáze dostávat 
	  náhodně nebo popořadě.

	- Jednoduché menu s možnostmi, např. věty náhodně/popořadě,
					     výběr barvy zobrazeného textu,
					     zobrazení všech vět,
					     výběr konkrétní věty.
